Get Started Today

Using the URP on a new project:

  • With the newly updated Unity Hub client, a new template with URP integrated has been added. Just select it, give the project a name, and click "create".

Upgrading an existing project in 4 easy steps:

  1. From the package manager window, find the Universal RP package and install it.
  2. Create a Pipeline Asset by going to Create → Rendering → Universal Render Pipeline → Pipeline Asset.
  3. Edit your project settings: go to Edit → Project Settings → Graphics and, under Scriptable Render Pipeline Settings, select the Pipeline Asset you just created.
  4. Upgrade your materials: go to Edit → Render Pipeline → Universal Render Pipeline → Upgrade Project Materials to UniversalRP Materials.
